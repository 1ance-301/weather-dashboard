# Weather Dashboard

## Website Purpose
A website to display weather of a city. It allows you to search for a city and then shows that city. It will also give you a five day forecast. It also saves your recent searches.

## Written With 
- HTML
- CSS
- Bootstrap
- jQuery
- JavaScript
